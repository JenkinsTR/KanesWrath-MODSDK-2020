3DS Max Setup

3DSMax 7 SP 1 or 3DSMax 8

Copy the contents of the 3dsMax7_Plugin folder into the installation directory of 3DSMax 7 or 8. You should not have 3DSMax running when doing this, and if you do, it will require you to restart 3DS Max to ensure the tools are set up properly.

3DSMax 9

Copy the contents of the 3dsMax9_Plugin folder into the installation directory of 3DSMax 7 or 8. You should not have 3DSMax running when doing this, and if you do, it will require you to restart 3DS Max to ensure the tools are set up properly.

Further Information

For further information on the tools installed by doing so, and their use, please check the Art Pipeline section of the documentation.
