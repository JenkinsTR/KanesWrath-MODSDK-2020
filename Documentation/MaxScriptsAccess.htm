W3D Scripts for Max

 

There are 3 ways to access these scripts

 

  1. There is a utility called “RTS Tools” in max.  To access the utility go to the utility tab and click “MAXscript” then select the “RTS Tools” in the dropdown menu.

 

  1. There is a Floater dialog box that can be docked.  To access the floater go to “RTS Tools” utility and click the Floater button.

 

  1. A custom Toolbar.  All the scripts can be added to a custom toolbar.  The following describes how to make a custom toolbar
