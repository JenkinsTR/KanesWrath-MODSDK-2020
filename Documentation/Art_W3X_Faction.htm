Faction Buildings

There are 5 states for a faction building:

  1. Build Up
  2. Pristine
  3. Damaged
  4. Really Damaged
  5. Rubble Collapsing

There are also a few extra objects for some of the factions buildings (these are explained in detail below)

  1. Upgrade Pieces
  2. (3) types of Windows (night, fire, glow). these are used very rarely.
  3. Gags (characters with animations that interact with the buildings)

General Rules When Creating Buildings 

  1. Use Alpha Test or Add, Do NOT use Alpha Blend
    1. If a transparent texture is being applied to a 2 sided quad then check the “Static Sorting Level” in the Material Surface Type rollout in the W3D Material.
  2. Keep # of meshes as low as possible.
    1. The build up state should have 5 meshes
    2. The Prisine state should have 1 mesh ( not including characters or windows )
    3. The D1 damage state should have 1 mesh ( not including the fire )
    4. The D2 damage state should have 5 meshes ( not including the fire )
  3. Planes should be 2 triangles
  4. Keep the vert counts as low as possible
  5. Try to Keep # of textures to <= 2
  6. Minimize the # of bones, drop bones in medium and low models
  7. Minimize overdraw. Geometry that’s never seen should be eliminated. However, Be careful not to delete geometry that may bee seen if the user rotates the camera.

Faction Structure Budget

Verts 2000 / 1250 / 750  (High / Medium / Low  LOD's)

Texture size 2x512

Max # meshes 2

Build Up Animation

For the build up animation the building will need to be separated into pieces that are individually animated.  Then a 1000 frame long animation of the pieces coming up out of the ground will need to be created.  The building should be cut into about 10 different pieces, each animated so that the animation only lasts for about 20% of the whole 1000 frames.  Then the animation for the pieces should be staggered to take place over the whole 1000 frame.

The geometry should be exported as <building name>_A.w3x

The skeleton should be exported as <building name>_ASKL.w3x

The animation should be exported as <building name>_ABLD.w3x

Pristine

The pristine version of the building.

This should be exported as <building name>.w3x

Damaged

The damaged state is the pristine version of the building with the damaged texture on it.

This should be exported as <building name>_D1.w3x

Really Damaged

The really damaged state is made up of 5 pieces: The main building, the front, left, back, right.

The 4 a,b,c,d pieces need to have their pivot points placed in the center of each piece, while the main building needs to have it’s pivot point placed at the origin.  Note the pivot points in the picture below.


This should be exported as <building name>_D2.w3x

Rubble Collapsing

This state is a crumbling animation of the left over main building from the Really Damaged state. Simply subdivide the main building into smaller pieces, closing off open sides, ensuring each piece has it's own pivot point, and then animate the pieces.

This should be exported as <building name>_D3.w3x

Upgrade Pieces

Faction Buildings will have extra objects that start out hidden and are faded on as the building upgrades are purchased in the game, these pieces are called "Upgrade Pieces". They should be separate geometry and have very specific names: V1, V2, V3.


Windows

There are 3 different types of windows that a Faction building can have: Night time, Fire, and Glowing. The windows are extra geometry that overlay the normal windows of the building. This extra geometry will allow for some rendering effects to be added to the buildings.


The names for the windows follow this convention:

  • WINDOW_G01 = Glow Window for a night time glow effect.
  • WINDOW_N01 = Night time window .
  • WINDOW_F01 = Fire Window for damaged states.

    • You should use a Default W3D shader and the blend mode needs to be Additive. By using Additive the black pixels of the texture will be transparent and the lighter/whiter pixels in the texture will brighten the geometry behind it.


    Gags

    Gags are characters with animations that interact with the buildings. These are only in Pristine version of the High LOD version of the building.
