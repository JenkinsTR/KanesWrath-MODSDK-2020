Creating a New Unit Portrait

The Sample MOD includes a new Unit Portrait for the Elven Warrior.  If you look at the Sample MOD MOD.xml file you will see the following include:

        <Include type="all" source="ART:Images/SampleUpdatedPackedImages.xml" />

This file should be located in the Art/Images folder of the MOD SDK.  If you open it up you will see the following:

        <!-- Packed Texture: PackedImages_219.tga -->
        <Texture id="PackedImages_220" File="PackedImages_220.tga" OutputFormat="A8R8G8B8" GenerateMipMaps="false"          AllowAutomaticResize="false"/>

        <PackedTextureImage id="Portrait_GDIElfWarrior" Texture="PackedImages_220" Rotated="false">
        <Dimensions x="128" y="128"/>
        <Coords x="0" y="0"/>
        <TextureDimensions x="128" y="128"/>
        </PackedTextureImage>        

The first tag references the actual image file, and lets the game know where that image is located.  It is then referenced by its ID in the PackedTextureImage tag.  The id of the PackedTextureImage is then used in the GDIElfWarrior.xml file under the GameObject tag:

            SelectPortrait="Portrait_GDIElfWarrior"
            ButtonImage="Portrait_GDIElfWarrior"

The trick is to make sure the xml file you are using to create these is located in the same folder as the art you are referencing.
