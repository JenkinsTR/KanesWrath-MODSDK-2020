Sample MOD

The SampleMOD is a simple modification to the game that does a few things to show the user how to use the MOD SDK in a basic manner.  It creates an AI personality, introduces a new unit, and updates the UI.  If you have never created a MOD before for any of our RTS games or would like a quick tutorial on how to use this MOD SDK, we strongly encourage you read the following sections.
