    Using FX Shaders As Materials

    Instead of W3D materials you can now use standard "DirectX 9 Shader" materials in Max, using the .fx shaders specifically written for the game.

    • Start Max, create a mesh as usual.

    • Create a new material for your mesh. As material select "DirectX 9 Shader" (see left image below):

    Setup the material (right image):

    • Select the FX file to use in the "DirectX 9 Shader" section. FX files to be used with the game have been put into the folder "3dsmax7\maps\fx\CnC3".

    • Select one of the rendering techniques from the drop down box "Technique" at the bottom of the dialog. (Ignore "Software Render Style" completely)

    • Depending on the shader, the "Parameters" section in the center will be generated.

    • A parameter can be:
      1. Light Source - The available light sources in the Max scene will be offered, or "Default Light". Note: Having one light source assigned to multiple parameters, will result in it having twice or more times the light contribution than the light actually has. For best previewing results, (directional!) light sources should be setup similar to the lights in WorldBuilder/in-game.
      2. Texture - The same rules as for other textures in the game apply to these textures, ie. they need to lie in the "Art/Textures" folder, can be TGA or DDS format etc...
      3. Color - A standard max color selection dialog, with decimal color values (0.0 to 1.0) is offered
      4. Decimal or Integer numbers - These edit fiels allow entering numbers. Sometimes vectors of numbers are needed by a shader. Then a row of four edit boxes will show up. Max can only display 1 or 4 boxes, however sometimes vectors have 2 or 3 components. We use the text next to the edit boxes to say "xy" or "xyz" then.

    • After setting up the material, export the mesh as usual to a W3X file.

    Note: Some FX shaders are animated, and can use the current time value for whatever purpose (color shift, texture coordinate animation etc). To preview a material animation use the timing bar and play/stop functionality in Max.

    If you are having problems with Max finding the FX shaders/textures in files that you open, make sure to add the "Art/Textures" and "3dsmax7\maps\fx\CnC3" directory to the "External Paths" in the 3DSMax "Customize/Configure Paths" dialog.
